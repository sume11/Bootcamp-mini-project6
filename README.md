# Express MVC Project
A simple Express.js server following the MVC pattern with the following routes:
- GET / → Home
- GET /about → About
- GET /contact → Contact
- GET /data/greeting → Dynamic greeting
- POST /data/submit → Accepts JSON and returns it
Run the server:
1. `npm install`
2. `node server.js`
Test endpoints using browser or Postman/Thunder Client.
